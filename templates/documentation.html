Model training parameters

Below are the descriptions of the model training parameters on the EDA page.

  • Total time to search for model: how much time in seconds is given for Auto-sklearn to search for a good model. In general, the more time is given, there is higher chance of finding a better model. See documentation
  • Per run time limit: Auto-sklearn tries to fit many models in the time it's given for training. This is the time it can use to optimize each of those models.
  • Folds for cv: x fold cross validation is used by this AutoML system to optimize hyperparameters.
  • Test set size (%): How much of the total available data should be used for testing. The test set will also be used for the feature importance module.

AutoML model results

Feature importance module

Permutation feature importance is calculated with n_repeats=10. Details of how it works at: sklearn.inspection.permutation_importance
